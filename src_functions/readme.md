POST /move

Saves an object of this shape in mongodb

```javascript

  game: {
    type: String,
  },
  board: [String],
  move: {
    cell: {
      type: Number,
      min: 0,
      max: 8,
    },
    value: {
      type: String,
      enum: ['o', 'x'],
    }
  }
}
```
